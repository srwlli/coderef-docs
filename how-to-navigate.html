How to navigate

This site is the front door to CodeRef — the place to arrive and orient. The docs are the map; the repo is the territory.

This docs site is the front door to CodeRef — the place to arrive and orient. A few framings for the reader.

A guide, not the truth

This site is a guide; the canonical sources are the truth. Every page here is composed from the owning domain's own words — CR-HTML dispatches each domain for its section. When you need the exact, authoritative detail, the page points you at the canonical source: a SKILL.md, a STANDARD, a CONTEXT.md.

The docs are the map; the repo is the territory.

Start here, then branch

Start at ASSISTANT — you're here. The entry section explains the whole ecosystem. From there:

  • CODEREF-CORE — the engine.
  • SKILLS — the 168 capabilities.
  • SESSIONS — how the daily loop runs.
  • TRACKING — how work is recorded.

The site grows with the ecosystem

Sections marked "coming soon" are real domains whose docs are being pulled from their owning agents. The nav shows the full intended shape from day one, so you can see where the system is going even before every section is filled in.

If you're an agent

This is your orientation, not your instruction set. Your actual rules are enforced mechanically — the gates — and documented canonically — the standards. This site helps you find them.
