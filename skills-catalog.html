The reference tier

Browse all skills

Every skill in the ecosystem, generated from the live registry at build time — filter by name or summary, narrow by category, and drill into a per-skill reference page.

This catalog is generated, not hand-written: at every build it is regenerated from TRACKING/skills.json (the registry view of the skill manifests on disk), so it always matches the corpus. Each entry links to a per-skill reference page with the skill's signpost, contract surface, and canonical source pointer.

Agents: the same data is published machine-readable at assets/skills-catalog.json.

No skills match the filter.
