ANALYSIS skill

/improve-skills

Read-only synthesizer that walks the kaizen.md substrate across all SKILL.md files, clusters observations, and emits a PROPOSED-EDITS report. Hermes pattern — never mutates source artifacts.

Slash command/improve-skills
CategoryANALYSIS
Version0.1.0
Statusactive
Runnerrun.mjs (deterministic mechanics ship with the skill)
Surfacesclaude, codex, agy

When to use

After kaizen substrate has accumulated for >7 days, to surface cross-skill drift patterns and propose corpus-wide ergonomics/correctness fixes. Trigger phrases: 'where is the kaizen drift', 'what's the cross-skill pattern', 'synthesize kaizen', 'propose skill improvements', 'improve skills'. Output is a markdown proposed-edits report; operator decides which proposals to action via /create-workorder.
